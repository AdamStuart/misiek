Interaction data

Interactions for each of the input species should be loaded in the SIF format. Each line should provide the following three fields for one particular interaction:
proteinLabel1 probability proteinLabel2
Where the probability is a real number ranging from 0 to 1.

Example:

fu i4 0.4724 fu_i4
fu i4 0.1622 fu_i11
fu i11 0.0654 fu_i11
fu i11 0.0801 fu_i12
fu i12 0.0803 fu_i12
