Affinity algorithm parameters

  • Number of iterations
  • Stop criterium - stops after n-th iterations without any center change
  • Preference - preference for node to be center (currently equal for every node)
  • Lambda

Affinity implementation parameters

  • Node name attribute
  • Edge name attribute (click refresh if you haven't edges attribute name on list)
  • Refine centers - if selected, refine method will be used after all iterations, for every cluster find greatest node to be center in this cluster
  • Take log - if selected, edges will be loaded to algorithm with Log(weigth)
