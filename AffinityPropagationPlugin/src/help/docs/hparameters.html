Affinity propagation parameters

  • Number of iterations
  • Stop criterion - stops after n-th iterations without any center change
  • Preference - preference for a node to be a center (currently equal for every node)
  • Lambda

Implementation-specific parameters

  • Node name attribute
  • Centers name attribute
  • Edge weight attribute (click refresh if no attribute names are listed)
  • Refine centers - if selected, a refinement procedure will be applied after all iterations have passed, to determining the best center for each cluster among all cluster members
  • Take log - if selected, edges weights will be set to log(edge_weight)
