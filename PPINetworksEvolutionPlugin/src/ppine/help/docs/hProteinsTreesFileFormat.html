Protein family trees

Protein family trees should be provided in an enriched standard recursive format. Each line in the file should describe one protein family tree as follows:
(speciesNode1, speciesNode2, ...)familyName;
Where speciesNode has recursive format:
speciesNode := [proteinNode1,proteinNode2, ...]speciesName
Where proteinNode has recursice format:
proteinNode := proteinName if proteins is a leaf, or:
proteinNode := (speciesNode1, speciesNode2, ...)proteinName.

Example:

([([ce_ce13562]ce,[dm_dm178]dm)bi_i1318]Bilateria,[sc_sc5512,sc_sc1833,sc_sc3895,sc_sc4997]sc)root_222;
